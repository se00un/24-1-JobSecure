진행 기간: 2024.05.09 - 2024.07.16
프로젝트 유형: 2024년 고용노동부 공공데이터 공모전 아이디어 기획 부문 제출작

고용노동부 공공데이터를 활용해 미래 일자리 미스매치 지역과 고용위기 가능 지역을 예측하고자 했던 프로젝트이다. 구인·구직·피보험자·실업급여 데이터를 시군구 및 직종 단위로 정리한 뒤, ARIMA와 LSTM을 활용해 시계열 예측을 수행하고, 예측 결과를 바탕으로 일자리 미스매치 지수를 계산했다. 또한 정부의 고용위기지역 지정 기준을 데이터 알고리즘으로 구현해 미래 위기 가능 지역을 식별하고자 했다. 이 프로젝트를 통해 처음으로 장기 시계열 데이터 전처리와 예측 모델링을 경험했으며 동시에 제한적인 변수만으로 복잡한 사회 문제를 예측하는 데에는 분명한 한계가 있다는 것을 파악했다. 또한, 데이터사이언스에서 가장 중요한 것은 *“왜 이 데이터와 모델을 사용하는가”*에 대한 근거 있는 설명이라는 점을 배웠다.
